# 紫微斗数排盘算法设计

> 版本：0.1.0 | 实现位置：`backend/app/core/chart_engine/`

## 1. 算法模块划分

| 模块 | 文件 | 职责 |
|------|------|------|
| 历法转换 | `calendar.py` | 公历↔农历、节气、时辰 |
| 宫位定位 | `palace.py` | 安命宫、身宫、十二宫 |
| 星曜安放 | `stars.py` | 十四主星、六吉六煞等 |
| 四化飞星 | `sihua.py` | 禄权科忌 |
| 大限 | `daxian.py` | 十年大运 |
| 流年 | `liunian.py` | 流年宫位 |

## 2. 输入

```python
{
    "birth_datetime": "1990-05-15T14:30:00",  # ISO 8601
    "gender": "male",                          # male / female
    "calendar_type": "solar",                  # solar / lunar
    "timezone": "Asia/Shanghai"
}
```

## 3. 输出

结构化 JSON，存入 `charts.chart_data` 字段（结构见 `database-schema.md`）。

## 4. 核心步骤（待阶段 1 实现）

### 4.1 历法转换
- 公历日期 → 农历年月日时
- 确定生年干支（年柱）
- 确定时辰（子丑寅卯…）

### 4.2 安宫
- 依生月、生时安命宫
- 依命宫安身宫
- 逆时针排列十二宫

### 4.3 安星
- 依生年天干安紫微星
- 依紫微位置安十四主星
- 安六吉星、六煞星等

### 4.4 四化
- 依生年天干定禄权科忌
- 标记各星四化属性

### 4.5 大限流年
- 依阴阳男女、命宫位置起大限
- 计算当前流年宫位

## 5. 流派说明

本项目默认采用 **三合派** 为基础，后续可扩展飞星派。
